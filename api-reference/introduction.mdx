```
https://app.aseed.ai/custdev
```

## API Overview

The Aseed Insight API is organized into the following sections:

- **Insight API** - Core functionality for managing insights, records, transcripts, and exports
- **Projects API** - Project management and project reports
- **Prompts API** - Managing prompts and templates
- **Presets API** - System and custom presets management
- **Notifications** - WebSocket-based real-time notifications

<Note>
Admin endpoints are excluded from this documentation. For administrative operations, please refer to the full OpenAPI specification or contact your system administrator.
</Note>
