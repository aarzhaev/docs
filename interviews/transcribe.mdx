## Overview
Transcripts are the foundation for all reports. Set the right options before you generate them.

## Step-by-step guide
<Steps>
  <Step title="Choose transcription options">
    Select the language and speaker settings that match your interview.
  </Step>
  <Step title="Start transcription">
    Begin transcription for one or more files in Project Records.
  </Step>
  <Step title="Wait for completion">
    Transcripts appear when processing finishes.
  </Step>
</Steps>

## Best practices
<Tip>
Use one language per file for the best accuracy.
</Tip>
