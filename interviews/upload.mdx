## Overview
Upload interviews to create transcripts and generate reports.

## Supported formats
- Audio: MP3, WAV, M4A, OGG
- Video: MP4, MOV, AVI, WebM
- Text: TXT, DOCX, PDF, MD, SRT, VTT

## Step-by-step guide
<Steps>
  <Step title="Open Project Records">
    Go to your project and find the **Project Records** section.
  </Step>
  <Step title="Upload files">
    Click **Upload Records** and select one or more files.
  </Step>
  <Step title="Confirm upload">
    Wait for the files to appear in the records list.
  </Step>
</Steps>

## Best practices
<Tip>
Use clear file names so you can quickly identify respondents later.
</Tip>
