This documentation covers everything you need to know about using Aseed for customer development research. Use this overview to quickly find the right guide for your task.

## Common workflows

### First-time user workflow

1. [Workflow](/workflow) - Understand the workflow
2. [Create a project](/projects/create) - Set up your first project
3. [Fill research context](/projects/context) - Define research goals and objectives (use AI to generate Research Context Brief or add manually)
4. [Upload records](/interviews/upload) - Add interview files
5. [Upload text transcripts](/interviews/text-transcripts) or [Transcribe records](/interviews/transcribe) - Add transcripts or generate from audio/video
6. [Create interview guide](/interviews/guide-builder) - Prepare interview structure
7. [Edit transcripts](/interviews/edit) - Review and correct if needed
8. [Generate single reports](/reports/single) - Create your first report
9. [Share or export](/sharing/public) - Share with stakeholders

### Advanced research workflow

1. [Create a project](/projects/create) - Set up your research project
2. [Fill research context](/projects/context) - Define research goals and objectives (use AI to generate Research Context Brief or add manually)
3. [Upload records](/projects/upload-records) - Add interview files to project
4. [Upload text transcripts](/interviews/text-transcripts) or [Transcribe interviews](/interviews/transcribe) - Add transcripts or generate from audio/video
5. [Create interview guide](/interviews/guide-builder) - Generate structured guides with AI and save to project
6. [Edit transcripts](/interviews/edit) - Review and correct if needed
7. [Batch generation](/projects/batch-generation) - Process multiple records at once
8. [Create a project report](/reports/project) - Synthesize insights across interviews
9. [Share or export](/sharing/public) - Share aggregated findings

### Text transcript workflow

1. [Create a project](/projects/create) - Set up your research project
2. [Fill research context](/projects/context) - Define research goals and objectives
3. [Upload text transcripts](/interviews/text-transcripts) - Upload and parse text files
4. [Create interview guide](/interviews/guide-builder) - Prepare interview structure
5. [Edit transcripts](/interviews/edit) - Review and correct if needed
6. [Generate single reports](/reports/single) - Create reports from transcripts
7. [Create a project report](/reports/project) - Aggregate insights
