## Overview
Project details help teammates understand the scope and purpose of your research.

## Step-by-step guide
<Steps>
  <Step title="Open the project">
    From the workspace, open the project you want to update.
  </Step>
  <Step title="Update project details">
    Edit the project name or description to reflect the current research focus.
  </Step>
  <Step title="Review your changes">
    Make sure the description stays concise and goal-focused.
  </Step>
</Steps>

## Best practices
<Tip>
Keep the description to one or two sentences so it stays easy to scan.
</Tip>
