## Overview
Project reports combine insights from multiple interviews into a single narrative.

## Step-by-step guide
<Steps>
  <Step title="Open Project Reports">
    In your project, go to the **Project Reports** section.
  </Step>
  <Step title="Choose structure">
    Select a preset or build a custom structure before generating.
  </Step>
  <Step title="Generate the report">
    Confirm the context and start generation.
  </Step>
</Steps>

## Best practices
<Tip>
Review single reports first so the synthesis is clean.
</Tip>
