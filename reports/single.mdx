## Overview
Single reports capture insights for one interview at a time. They are the building blocks for project-level synthesis.

## Step-by-step guide
<Steps>
  <Step title="Open Generate Reports">
    In Project Records, click **Generate Reports**.
  </Step>
  <Step title="Choose report types">
    Select the report types you need for each interview.
  </Step>
  <Step title="Start generation">
    Confirm and generate the selected reports.
  </Step>
</Steps>

## Report types
- Transcript
- Q&A
- General
- JTBD
- Persona
- Custom

## Best practices
<Tip>
If you are unsure, start with Transcript and Q&A, then expand.
</Tip>
